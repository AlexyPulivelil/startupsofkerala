## Run Locally

Clone the project

```bash
  git clone https://github.com/lemokami/startupsofkerala
```

Go to the project directory

```bash
  cd startupsofkerala
```

Install dependencies

```bash
  npm install 
```


Start the dev server

```bash
  npm run dev
```
